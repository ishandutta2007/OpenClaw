## Why FasterClaw?

While OpenClaw is powerful, FasterClaw focuses on speed and efficiency.

- ü™∂ **Ultra-lightweight**: Less than 10MB memory footprint ‚Äî 99% smaller than OpenClaw.
- üí∞ **Minimal Cost**: Runs on $10 hardware ‚Äî 98% cheaper than a Mac mini.
- ‚ö°Ô∏è **Blazing Fast**: 400x faster startup time, booting in under a second on a single 0.6GHz core.

1. **Gateway** ‚Äî local control plane routes messages, manages sessions & memory  
2. **LLM Brain** ‚Äî powerful model (e.g. Claude Opus/Sonnet) reasons & plans  
3. **Tools & Skills** ‚Äî execute actions (read/write files, send messages, run commands‚Ä¶)  
4. **Channels** ‚Äî talk to your assistant from any supported messaging app  
5. **Heartbeat** ‚Äî periodic proactive checks & notifications
